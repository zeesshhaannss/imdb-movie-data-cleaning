# IMDB Movies Data Cleaning Project

## Overview
This project demonstrates cleaning and preprocessing of an IMDB movies dataset in Excel, preparing it for analysis and pivot table summarization.

## Tools Used
- Excel

## Data Cleaning Steps
1. **Loaded Data:** Imported dataset using Excel 'Get Data'.
2. **Column Names:** Standardized column names for readability.
3. **Release Year:** Standardized date formats and extracted year into a new column.
4. **Date Errors:** Checked and corrected inconsistent or invalid dates.
5. **Genre Column:** Combined multiple genre columns into a single column using TEXTJOIN.
6. **Blanks:** Checked and removed blank cells across columns.
7. **Duration Column:** Removed alphabetic characters and standardized empty or invalid values.
8. **Country Column:** Corrected country names and standardized spellings.
9. **Income Column:** Removed currency symbols, letters, and spaces; converted to numeric.
10. **Votes Column:** Removed dots and converted to numeric.
11. **Score Column:** Removed unnecessary characters and standardized values.
12. **Duplicates:** Identified and removed duplicate records.
13. **Table Format:** Converted the dataset into a structured Excel table.
14. **Spelling Check:** Used Excel's built-in spell check and standardized text capitalization.
15. **Outlier Check:** Identified numeric outliers (e.g., a movie with 576 income and 668,473 votes).
16. **Text Standardization:** Applied proper case to text columns for consistency.
